Quizzer App

A simple Python-based quiz application built using Tkinter. The app  allows users to take quizzes, view results and save data locally for tracking performance. Designed with a user-friendly interface and structured for modular development.

Features
 ▪ Multiple-choice questions with radio buttons
 ▪ Timer with countdown and tracking
 ▪ Result calculation andd score display
 ▪ Pie chart visualization of performance (using matplotlib)
 ▪ Save/Load results and quizes using JSON
 ▪ Media/ Image support via Pillow
 ▪ Styled widgets
 ▪ Organized folder structure for assests and data

Folder Structure
 ▪ /assests - Images and other media files
 ▪ /data - JSON files for questions and saved results
 ▪ /docs - Documention files
 ▪ /src
 ▪ main.py - Application script, double click to run
 ▪ README.md - Project documentation
 ▪ LICENSE - Licensing information
 ▪ requirements.txt - Python dependencies

Requirements

Install dependencies with:

bash
pip install -r requirements.txt
